# IRSMAT-2.0

Игра для математических уравнений для детей 1-7  класс ОБНОВЛЕНИЕ

Основные улучшения игры

Вот что было добавлено и изменено для повышения интереса:

Поиграть в нашу игру теперь можно через браунзер пк/телефон через ссылку на GITHUB: https://percoset2033.github.io/IRSMAT-2.0/

Для андроида есть APK FILE: https://disk.yandex.ru/d/Gzm7gWeiDT0OhA

Коллекция наклеек (20 штук): Иконки цветочка (fa-flower) и птички (fa-dove) теперь корректно отображаются. Добавлены новые наклейки для разнообразия коллекции.

Усложнённое получение наклеек: Теперь их нельзя получить за каждый ответ.

За серию правильных ответов: После 3 правильных ответов подряд есть 50% шанс получить случайную наклейку.

За прохождение уровня: Даётся одна гарантированная наклейка.

В магазине: Появился новый товар — «Набор наклеек» за 20 монет, который даёт гарантированную случайную наклейку.

Контент до 7 класса: Добавлены примеры для 5, 6 и 7 классов (дроби, проценты, пропорции, простые уравнения), что расширяет обучающий материал.

Улучшенные подсказки: Текст подсказок теперь содержит краткое объяснение решения (например, "2x = 4, x = 2"), что помогает понять логику.

Новая мотивация: Добавлена секретная награда — «Платиновая корона», которая появляется, когда собраны все 20 наклеек.
